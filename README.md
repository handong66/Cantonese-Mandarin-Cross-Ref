# 粤普对照查询工具 (Cantonese-Mandarin Cross-Reference Tool)

这是一个交互式的粤语（粤拼）与普通话（拼音）双向对照查询工具，旨在帮助学习者直观地理解和掌握两种语言之间的语音对应规律。

## 项目背景

本项目是基于 [香港教育大学 中台港常用字粵普聲韻調交互對照表](https://ckc.eduhk.hk/ckc/can/) 进行修改和优化的**粤拼 (Jyutping) 版本**。在原有的对照基础上，将粤语拼音统一为更符合现代学术和学习习惯的粤拼方案，并增加了音频播放及移动端适配等功能。

## 主要功能

- **双向查询**：支持“普通话 → 粤语”及“粤语 → 普通话”两种查询模式。
- **全方位对照**：涵盖声母、韵母、声调的完整对应关系。
- **汉字检索**：支持通过汉字直接搜索，快速定位其在两种语言中的读音。
- **音频播放**：内置粤拼音频，点击示例汉字即可聆听标准粤语发音。
- **简繁切换**：提供简体和繁体两个版本，满足不同地区用户的使用习惯。
- **响应式布局**：针对手机和平板进行了专门的布局优化，方便随时随地学习。

## 使用方法

1. **选择模式**：在页面顶部的下拉菜单中选择查询方向（默认为“普通话 → 粤语”）。
2. **浏览列表**：
   - 在左侧（移动端为上方）的列表中点击您想要查询的声母、韵母或声调。
   - 右侧（移动端为下方）将立即显示与之对应的另一方语音规律。
3. **查看示例**：
   - 对应关系下方会列出具有代表性的示例汉字。
   - **点击汉字**：即可播放该汉字的粤语发音。
4. **汉字搜索**：在搜索框中输入单个汉字，系统将自动跳转并高亮显示该汉字的对应规律。
5. **版本切换**：点击页面右上角的“繁/简”按钮，即可在繁体中文和简体中文界面之间切换。
